# Changelog

All notable changes to Authinator will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- JWT-based authentication with access and refresh tokens
- SSO integration with Google, Microsoft, Auth0, and Okta
- Service registry for microservices architecture
- User approval workflow for admin oversight
- Multi-tenant customer isolation
- React TypeScript frontend with modern UI
- Service directory page for easy navigation between registered services
- Management command for SSO provider setup
- Custom authentication adapters for auto-signup
- Comprehensive API documentation

### Security
- Environment-based secrets management
- No hardcoded credentials in codebase
- Secure session handling for OAuth flows
- JWT token validation

## [0.1.0] - 2025-02-26

### Added
- Initial release of Authinator
- Core authentication functionality
- Basic user management
